 LOL 云顶之弈 自投脚本

### 版本更新
##### v1.3
1. 加入随机放装备  
2. 优化终端显示效果  
##### v1.2 
1. 修复了大家都比我早投降时不会开局的bug

##### v1.0
云顶之弈自动开局，3-4 自动投降  
刷代币用，有几率被检测消极游戏封号 1 天  
建议排位使用

## 使用方法
1. 电脑分辨率设置为 1920*1080，100%缩放
2. LoL 游戏内分辨率 1920*1080，窗口化
3. lol 进入云顶之弈房间
4. 用 ***管理员模式*** 启动dist文件夹中的 `v1.0.exe`
5. 客户端位置不要动
