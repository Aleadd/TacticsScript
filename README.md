# TacticsScript
 LOL 云顶之弈 自投脚本

## v1.0

云顶之弈自动开局，3-4 自动投降  
刷代币用，有几率被检测消极游戏封号 1 天  
建议排位

## 直接使用的设置

1. 电脑分辨率设置为 1920*1080，100%缩放
2. LoL 游戏内分辨率 1920*1080，窗口化


## 使用方法

1. lol 进入云顶之弈房间
2. 用 ***管理员模式*** 启动dist文件夹中的 `main.exe`
